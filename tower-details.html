Tower

Project information

  • Category: Promotions and Listings
  • Project date: 07 May, 2022
  • Project URL: https://tower-dcade.herokuapp.com

Tower Events is an app developed for the listing of public and private events. It serves as a proof of concept for a social-media style, web-based tool. The principle challenge of Tower was not the code itself but the imperative to build it in only 4 days, fully production ready and after 11 days of non-stop project work beforehand.

Tower uses Node.js for its back-end, and MongoDB as both database hosting and as an object relational mapper (ORM). The front is built in Vue3 and JavaScript, using keyframe animations and SCSS for style.
