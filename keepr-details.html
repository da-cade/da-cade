Keepr

Project information

  • Category: Social Media
  • Project date: 05 June, 2022
  • Project URL: https://keepr-dac.herokuapp.com/

Essentially a Pinterest clone, Keepr is a social media site where users can share and save images to particular "Vaults". Keepr was designed strictly from a designer's mock-up, indicating my ability to produce applications in a professional setting. Keepr adheres not only to this mock but to the guidelines of accessibility and clean code.

This program also demonstrates my increased comfort with Vue3 and C#. Using Vue, I employ Transition and Keep-Alive functionality, the first for styling and the second for memory. Additionally, I developed upon my experiments with the Vue Router in AllSpice, experiments which have contributed greatly to my progress with React.js Router. Keepr uses MySQL for data storage and Dapper as an ORM.
