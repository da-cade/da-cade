AllSpice

Project information

  • Category: Tools and Experimentation
  • Project date: 30 May, 2022
  • Project URL: https://allspice.herokuapp.com/

AllSpice is an app designed for the tracking and potential sharing of personal recipes and pictures. It was developed over the course of 4 days, and uses many experimental concepts in the Vue3 JavaScript framework, such as a feature replacement for 'content-editable' to allow for multiple edits to be performed towards personal content.

AllSpice was my first program using C# for the back-end, an extremely concise and swift component-oriented, object-oriented language. AllSpice uses MySQL for database management and Dapper as an Object Relational Mapper (ORM).
