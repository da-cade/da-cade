Plan-It: Agile Sprints and Projects

Project information

  • Category: Tooling
  • Project date: 06 May, 2022
  • Project URL: https://planit-scrum.herokuapp.com

Plan-It is essentially a rudimentary clone of Trello. The app, built with another developer by practicing peer-programming, is meant to offer a visual aid for the breaking-up of projects into sprints and tasks. The UI utilizes reactive elements, and the program can be modified to accomodate multiple users.

Plan-it served as my introduction to Vue3. It utilizes a beginner-level understanding of Vue3, and was a transition from the simpler Options API to the Composition API. Plan-It has a Node.js back-end and Mongoose/MongoDB for a database and ORM
